JS - C 02 IF

1- IF...

la condition if est l'une des plus utilisées et est également la plus simple à appréhender puisqu'elle va nous permettre d'éxecuter un bloc de code SI et seulement SI le résultat vaut true.

2- IF éxecuté dans le if.js

  • if(x > 3){
    alert('x contient une valeur supérieure à 1');
    }
  • if(x == y ){
    alert('x et y contiennent la même valeur');
    }

! 3éme if : tout test d'une condition est évalué dans un contexte booléen ; donc à true à l'exception des valeurs suivantes qui vont être évaluées à false.

    Le booléen false :
  • La valeur 0
  • Une chaîne de caractère vide
  • La valeur null
  • La valeur undefined
  • La valeur NaN (Not a Number)

3- Inverser la valeur logique d'un test

Dans une condition if, le code n'est exécuté que si le résultat de la comparaison est true.

Dans certaines situations on préfère vérifier si la comparaison est false.

On peut vérifier que c'est faux en comparant explicitement le résultat de la comparaison à false
