JS - III 09 Boucles

1- Présentation des boucles

Les boucles vont nous permettre d'éxecuter plusieurs fois un bloc de code, c'est a dire d'éxecuter un code en boucle tant qu'une condition donnée est vérifiée ; utiliser une boucle nous permet d'écrire le code a exécuter qu'une seule fois.


Il existe 6 boucles en JS :

  1. La boucle while = " tant que"

  2. La boucle do-while = " faire tant que"

  3. La boucle for = " pour..."

  4. La boucle for-in = " pour...dans..."

  5. La boucle for-of = " pour... parmi"

  6. La boucle for-await = " pour (en attente)... parmi"

Le fonctionnement général des boucles est toujours le même : on pose une condition, généralement liée à la valeur d'une variable et on exécute le code de la boucle « en boucle » tant que la condition est vérifiée.
Pour en pas rester bloqué à l'infini dans une boucle, il faudra que la condition donnée soit fausse à un moment donné (pour sortir de la boucle).
Pour que notre condition devienne fausse à un moment, on pourra par exemple incrémenter ou décrémenter la valeur de notre variable à chaque nouveau passage dans la boucle (ou modifier la valeur de notre variable selon un certain schéma).



Les boucles sont essentiellement composées de 3 choses :


  1. Une valeur de départ qui sert à intialiser la boucle et qui servira de compteur.
  2. Un test ou une condition de sortie qui précise le critére de sortie de la boucle.
  3. Un "itérateur" qui va modifier la valeur de départ de la boucle à chaque passage de la boucle jusqu'à ce que la condition de sortie soit vérifiée. Pour cela on devra "incrémenter" la valeur de départ.

2- Les opérateurs d'incrémentation et de décrémentation

Incrémenter = Ajouter 1 à une valeur

Décrémenter = Enlever 1 à une valeur

Exemple Résultat
++x Pré-incrémentation : incrémente la valeur contenue dans la variable x, puis retourne la valeur incrémentée
x++ Post-incrémentation : retourne la valeur contenue dans x avant incrémentation, puis incrémente la valeur de $x
--x Pré-décrémentation : décrémente la valeur contenue dans la variable x, puis retourne la valeur décrémentée
x-- Post-décrémentation : retourne la valeur contenue dans x avant décrémentation, puis décrémente la valeur de $x

3- La boucle "while"

La boucle while, "tant que" en français va nous permettre de répéter une série d'instruction tant qu'une condition donnée est vraie, c'est à dire tant que la condition de sortie n'est pas vérifiée.




4- La boucle do... while

La boucle do.. while "faire... tant que" est relativement semblable à while, la différence réside dans l'ordre dans lequel vont se faire les opérations.
On effectura un premier passage dans une boucle do...while même si la condition de sortie n'est pas vérifiée, donc le code sera exécuté au moins une fois.
