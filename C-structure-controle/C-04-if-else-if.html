1- if...else... if ...else en JS

La condition if...else if... else (si... sinon... sinon) est une structure conditionelle encore plus complète puisqu'elle permet de générer et de prendre en charge autant de cas que l'on souhaite.

Nous allons pouvoir écrire autant de else if que l'on veut dans cette conditio.

if ( x > 1) {
alert('x contient une valeur strictement supérieur à 1');
} else if ( x == 1){
alert('x contient la valeur 1');
} else {
alert('x contient la valeur strictement inférieur à 1');

Dès qu'un test va être validée le HS va ignorer les tests suivants. Il faut donc faire bien attention à l'ordre des else if lors de l'écriture d'une condition pour obtenir le résultat souhaité.
