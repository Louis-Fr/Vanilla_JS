1- if - else

La structure conditionelle if...else (en français si...sinon) sera plus complète var on va exécuter un bloc de code si un test renvoie trueou un autre dans le cas contraire.

If... else...

  • if(x > 1){
    alert('x contient une valeur supérieure à 1');
    }else{
    alert('contient une valeur infèrieur à 1'); }
