JS - C 01 Les structures de contrôle

1- Les conditions et les boucles

Il existe deux types de structures de contrôle de base, que l'on retrouve dans tous les langages de progammation et donc en JS.

  • Les structures de contrôle conditionelles : les conditions
  • Les structures de contrôle de boucle : les boucles

Les conditions vont nous permettre d'éxécuter un certain nombre d'instruction si et seulement si une certaine condition est vérifiée

Les boucles vont nous permettre d'exécuter un bloc de code en boucle tant qu'une condition donnée est vérifié.

A- Présentation des conditions

  • La condition if (si);
  • La condition if... else (si... sinon)
  • La condition if... else... else (si... sinonsi...sinon)
