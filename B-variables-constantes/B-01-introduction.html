JS - B 01 - Introduction

1 - Qu'est-ce qu'une variable?

Une variable est un conteneur servant à stocker des informations de manière temporaires, du texte (une chaîne de caractère) ou un nombre. La variable varie !

Elle stocke différentes valeurs au fil du temps.

Une variable et la valeur sont deux choses différentes

2 - Règles de déclaration

On dit (comme en php) que l'on "déclare" une variable. On déclare une variable avec les mots clefs :

  • let
  • var
  • Le nom d'une variable doit obligatoirement commencer par une lettre ou un underscore_
  • Le nom d'une variable ne doit contenir que des lettres
  • il n'y a pas d'espaces
  • cf. introduction.js

3- Initialiser une variable

quand la variable est vide on dit que l'on stocke une valeur ou qu'on initialise une variable

    La portée d'une variable désigne l'endroit ou cette variable va pouvoir être utilisée dans un script

    Nous reparlerons donc de cette portée des variables quand nous aborderons les fonctions en JavaScript

  • D- Conclusion : Let ou var

    La syntaxe de déclaration des variables avec let correspond à la nouvelle syntaxe, var est l'ancienne syntaxe, vouée à disparaître

    Vous devez donc utiliser le mot clef let pour déclarer vos variables, même si c'est plus contraingnant que var

5- Quelle utilité pour les variables

Il sera pratique de stocker différents types de données ou d'informations dans une variable

Exemple avec la fonction prompt() on récupère une donnée saisie par l'utilisateur, puis on fait script avec une variable qui stockera la donnée quand elle sera envoyée.
