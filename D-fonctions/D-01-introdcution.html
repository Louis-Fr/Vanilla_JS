JS - IV 01 Fonctions, introduction


Les fonctions en JS sont incourtournables ! En voici deux exemples et une définition.

1- Les fonctions prédéfinies

Une fonction est un bloc de code nommé et reutilisable et dont le but est d'éffectuer une tâche précise, les fonctions possèdent de nombreux atouts.

Il existe des fonctions "prédéfinies" tel que random qui est une fonction mathématique

Le nomd'une fonction est suivi d'un couple de parenthèse qui contiennent le plus souvent des arguments ex. replace() le premier argument de replace() est la chaîne de caractères que l'on cherche et le second est celle qui va la remplacer

Récapitulatif sur les fonctions

Les fonctions sont des blocs de code nommés et réutilisables et dont le but est d'effectuer une tâche précise ;

  • Il existe 2 types de fonctions en JS : les fonctions natives ou prédéfinies (ou des méthodes) qu'on n'aura qu'à appeler et les fonctions personnalisées qu'on va créer
  • Pour exécuter le code d'une fonction il faut l'appeler. Pour cela, il suffit d'écrire son nom suivi de parenthèses en passant éventuellement des arguments dans les parenthèses.
